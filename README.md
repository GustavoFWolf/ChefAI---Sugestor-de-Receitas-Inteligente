# ChefAI---Sugestor-de-Receitas-Inteligente
O intuito do projeto é criar um sugestor de receitas rápido, onde o usuário pode informar os ingredientes e suas respectivas quantidades aproximadas,
e o sugestor gerará três receitas simples de no máximo 30 minutos, o sugestor também apresenta filtros onde posso escolher receitas saudáveis, vegetarianas,
sem lactose, entre outros.
Esse projeto me permitiu praticar conceitos de POO em Java como herança,abstração e tratamento de erros com blocos try/catch. Além de estar integrada como uma API dde LLM,
nesse caso foi usado o Langchain4j junto com o Ollama, uma escolha segura e gratuita e foi escolhido a ferramenta Maven para gerenciar as dependências.
